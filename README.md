# 작업 하는 방법

1. 이 Repository를 클론받는다
 - "git clone https://github.com/wooogi123/koreahacks2020.git"
2. 각자 작업할 새 branch를 생성한다
 - "git checkout -b <사용할 브랜치 이름(화살표 괄호는 제외)>"
3. 작업한 후 작업한 내용을 commit 한다.
 - "git add ."
 - "git commit -m <커밋 메세지 내용(화살표 괄호는 제외`)>"
4. 저장소에 push 한다.
 - "git push"
